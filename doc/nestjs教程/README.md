# NestJS 教程

## 概述

NestJS 是一个用于构建高效、可扩展的 Node.js 服务器端应用程序的渐进式框架。它使用 TypeScript 构建，并结合了 OOP（面向对象编程）、FP（函数式编程）和 FRP（函数式响应式编程）的元素。

本教程将从最基础的 NestJS 概念开始，逐步深入到高级特性，包括安全、部署、微服务等，帮助你全面掌握 NestJS 框架。

## 学习目标

通过本教程的学习，你将掌握：

- NestJS 的核心概念和架构设计
- 如何使用 NestJS 构建 RESTful API
- 如何处理数据库、认证、授权等常见后端需求
- 如何优化和部署 NestJS 应用
- 如何构建 NestJS 微服务

## 适合人群

- 前端开发者想要学习后端开发
- 后端开发者想要学习 NestJS 框架
- 想要了解现代 Node.js 框架的开发者
- 想要构建高效、可扩展的 Node.js 应用的开发者

## 前置知识

- 基本的 JavaScript/TypeScript 知识
- 基本的 Node.js 知识
- 基本的 HTTP 协议知识
- 基本的数据库知识

## 学习建议

1. 按照教程的顺序逐步学习，不要跳过任何章节
2. 每学习一个章节，尝试自己动手实践
3. 阅读官方文档，深入理解概念
4. 参与社区讨论，解决遇到的问题
5. 尝试构建自己的 NestJS 应用

## 版本说明

本教程基于 NestJS v11.x 版本编写，使用 TypeScript 作为开发语言。
