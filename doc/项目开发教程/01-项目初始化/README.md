## 项目概述

本教程将带领你从零开始构建一个完整的后端服务项目，基于 NestJS 框架，包含以下核心功能：

- 用户认证系统
- 权限管理
- 路由管理
- 用户管理
- 角色管理
- 日志管理
- 文件上传
- 数据字典
- 通知公告
- 邮箱服务

项目采用模块化设计，易于扩展和维护，适合作为各种 Web 应用的后端服务。

## 技术栈

- **框架**：NestJS v11.x
- **语言**：TypeScript
- **数据库**：MySQL、SQLite
- **ORM**：TypeORM
- **认证**：JWT、Passport
- **API 文档**：Swagger、Knife4j
- **文件上传**：Multer
- **WebSocket**：Socket.io
- **日志**：Winston
- **部署**：Docker、Zeabur
