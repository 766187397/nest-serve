# 项目开发教程

## 概述

本教程将带你从零开始构建一个完整的后端服务项目，基于 NestJS 框架，包含用户认证、权限管理、日志管理、文件上传等核心功能。

通过本教程的学习，你将掌握如何使用 NestJS 构建一个生产级别的后端服务，包括项目初始化、核心功能开发、扩展功能开发、项目优化与部署等各个环节。

## 学习目标

通过本教程的学习，你将掌握：

- 如何进行项目需求分析和技术选型
- 如何设计合理的项目结构
- 如何搭建 NestJS 基础框架
- 如何开发核心功能模块
- 如何开发扩展功能模块
- 如何优化和部署项目
- 如何进行项目监控与维护

## 适合人群

- 想要学习如何构建完整后端服务的开发者
- 想要学习 NestJS 项目实战的开发者
- 前端开发者想要了解后端开发流程
- 后端开发者想要提升项目开发能力

## 前置知识

- 基本的 JavaScript/TypeScript 知识
- 基本的 Node.js 知识
- 基本的 HTTP 协议知识
- 基本的数据库知识
- 基本的 NestJS 框架知识（建议先学习 NestJS 教程）

## 学习建议

1. 按照教程的顺序逐步学习，不要跳过任何章节
2. 每学习一个章节，尝试自己动手实践
3. 阅读相关文档，深入理解概念
4. 参与社区讨论，解决遇到的问题
5. 尝试在现有项目基础上添加新的功能

## 项目特点

- 模块化设计，易于扩展和维护
- 完整的用户认证和权限管理系统
- 完善的日志管理系统
- 支持文件上传和大文件分片上传
- 支持 WebSocket 实时通信
- 支持定时任务
- 支持多种数据库（MySQL、SQLite 等）
- 支持 Docker 容器化部署
- 完善的 API 文档

## 技术栈

- **框架**：NestJS v11.x
- **语言**：TypeScript
- **数据库**：MySQL、SQLite
- **ORM**：TypeORM
- **认证**：JWT、Passport
- **API 文档**：Swagger、Knife4j
- **文件上传**：Multer
- **WebSocket**：Socket.io
- **日志**：Winston
- **部署**：Docker、Zeabur
